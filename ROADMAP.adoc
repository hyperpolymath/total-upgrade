== Roadmap

=== Phase 1: Template Completion

==== Missing RSR Compliance Files

* [ ] Create `.well-known/security.txt` (RFC 9116)
* [ ] Create `.well-known/ai.txt` (AI crawler policy)
* [ ] Create `.well-known/humans.txt` (credits file)
* [ ] Create `RSR_COMPLIANCE.adoc` specification document
* [ ] Create `guix.scm` or `flake.nix` package definition

==== State Management

* [ ] Create `STATE.scm` Guile Scheme lifecycle tracker
* [ ] Create `META.scm` project metadata
* [ ] Implement state-phase recipes in justfile

==== Template Placeholder System

* [ ] Document all `{{PLACEHOLDER}}` values
* [ ] Create `init.sh` or `just init` recipe for placeholder replacement
* [ ] Add validation for unreplaced placeholders in CI
